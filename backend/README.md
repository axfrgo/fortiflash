# FortiFlash Backend

Backend API for FortiFlash - Video Watermark Remover & Enhancer.

## Stack
- Python 3.11+
- FastAPI
- Uvicorn
- Video processing libraries (to be integrated)

## API Endpoints
- `GET /` - Health check
- `POST /remove-watermark/` - Remove watermark from video
- `POST /enhance-video/` - Enhance video quality
